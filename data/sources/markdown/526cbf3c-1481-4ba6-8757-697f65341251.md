# 一、 总则

## 1.1 目的

为明确项目人员考核办法，有效激励交付中心项目人员的工作积极性，并且通过项目考核实现“规范项目管理、提高项目质量、提升客户满意度、实现客户增值价值、控制项目成本”的目标，特制定本考核与激励制度。
## 1.2 适用范围

万户软件自主交付的项目。
# 二、 名词解释

 财务工时：项目系统最终统计出的项目财务工时。
 预算人天：指项目经理根据项目合同要求制定的项目计划所确定需要投入项目天数。
 项目毛利：指项目合同额 - 采购成本 - 交付成本预算 - 伙伴分成 - 商务费用等成本后的利润金额。
 预算管理委员会：由各业务单元和交付中心相关人员组成。
# 三、 项目考核及执行流程

## 3.1 考核责任人

预算管理委员会参与预算的审批，交付中心负责组织项目的考核工作，人力资源部负责组织项目奖金的计算和发放工作。
## 3.2 执行过程

1. 毛利率阶段，销售发起毛利率评估流程，项目经理根据相关文件预算交付成本并由PMO和业务单元审核确认；
2. 合同签订后，销售发起销售额明细划分流程，由PMO根据预算情况和项目实际情况确定最终的交付成本，并经交付中心、营销中心和相关业务单元负责人审批确认；
3. 依据合同额和审批通过的项目交付成本，计算出项目毛利并确定项目奖金预算；
4. 项目立项启动后，由项目经理发起预算流程，预算管理委员会审批确定预算；预算值超过估算值20%的项目预算审批时需扩大预算管理委员会成员范围；
5. 由项目经理制定工作计划、分配工作任务、确定项目绩效目标，并明确每项工作任务奖金所占奖金的比例；
6. 项目验收后，由PMO负责组织对项目进行考核，由项目经理负责组织对项目团队成员进行考核，考核结果由项目管理委员会审核；
7. 项目经理负责将项目考核成绩和项目奖金分配比例告知项目团队；
8. 如项目团队成员对项目考核成绩有异议，可向PMO提起申诉，PMO负责组织复议并给出最终决定；
9. 项目考核和项目奖金发放均以单体项目为单位进行计算和发放，逐一累加；
10. 如员工在当年离职或被辞退，则该年度个人奖金不予发放。
# 四、 考核成绩评定

## 4.1 考核标准

项目考核成绩分为优秀、满意、合格、待改进和不合格五类。项目考核成绩由项目交付成本决定。项目考核得分如下：
项目考核得分 = （1 − 成本超支率）*100%

| 评定结果 | 考核分数T | 项目绩效系数 |
| --- | --- | --- |
| 优秀 | T >=100 | 1.1 |
| 满意 | 95 <= T < 100 | 1 |
| 合格 | 85 <= T < 95 | 0.8 |
| 待改进 | 70 <= T < 85 | 0.5 |
| 不合格 | <70 | 0 |

项目评定结果为不合格，则无项目奖金。
项目交付过程中，若出现质量安全事故（数据丢失、安全事件、涉密事件、重大客诉）等红线事件，该项目的评定结果不得高于0.5。
## 4.2 成本考核评分

成本考核主要考核项目决算成本与预算成本的偏差。
成本超支率计算公式如下：决算成本−预算成本预算成本决算成本 = 决算人工成本+决算差旅成本+外包成本+其他项目投入成本，决算人工成本指为本项目工作的所有人员的财务工时成本之和，每个人员的财务工时成本为财务工时天数×员工职级对应的人天成本。
员工职级对应的人天成本每年由人力资源部门和财务部门测算后发布。
预算成本=预算人天*1000元+预算差旅成本+外包成本。
## 4.3 项目需求变更

对于客户提出的不属于合同交付范围内的需求和其他重大变更，项目经理可提交《客户需求变更审批流程》，评估交付成本及影响分析，经交付中心、营销中心、业务单元等部门审批通过后执行。
经审批确认的变更成本，不计入项目的决算成本。
# 五、 奖金计算办法

## 5.1项目奖金包

项目奖金包采用预算制方法，依据项目的毛利计算。
项目奖金包 =  项目预算毛利*奖金系数。
项目预算毛利 = 合同额 −采购成本 − 交付成本预算 − 伙伴分成 − 商务费用。最终数据来源于销售提交的销售额划分明细表审批流程和销售额划分明细表变更审批流程 。
 交付成本预算是指在销售额划分阶段预算确定的交付成本，包含为本项目投入的所有人力成本预算、差旅成本预算、外包成本之和。
 人力成本预算是指在销售额划分阶段预算的为项目投入的人天数*1000元。
奖金系数：依据项目级别进行确定，详见下表。产品原型项目、战略项目等项目属性，在营销过程中、最迟在项目启动之前由公司确认。项目级别定义请参考附件《项目定级分值表》。

| 项目类别 | 奖金系数 |
| --- | --- |
| 战略项目 | 6% |
| A类项目 | 5% |
| B类项目 | 4% |
| C类项目 | 4% |
| D类项目 | 4% |
| E类项目 | 3% |
| 万相公文项目 | 1% |
| 无纸化会议项目 | 1% |

注：
1、 对于在项目启动之前就明确亏损的项目，制定单个项目的激励方案，报总经办会议审批后执行；
2、 非战略项目，若项目决算亏损，则无项目奖金。
## 5.2项目奖金核算

1、当项目决算成本小于预算成本，即项目节省了交付成本时，按以下公式计算项目奖金。
项目奖金 = 项目预算毛利*奖金系数*项目绩效系数 +（项目预算成本−项目决算成本）*奖金系数*5 2、当项目决算成本大于等于预算成本，即项目成本超支时。
项目奖金 = 项目奖金包 *项目绩效系数项目组成员奖金分配比例由项目经理提交，PMO审批。
## 5.3项目奖金发放

项目奖金分两次发放。第一次为项目结项且向运维交接完成后，按有效到款的同比例发放；第二次为验收款回款之后发放剩余部分。
发放流程为由项目经理申请，交付中心、业务单元和客户服务中心审核后，按季度汇总发放。
若项目在交接运维时，还有属于交付范围内的遗留工作，需针对遗留问题的工作量进行估算，并把估算工作量合并到项目交付成本里。
