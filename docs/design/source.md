# 外部知识输入

外部知识输入是知识大脑感知外部世界的入口。

知识大脑不能只依赖用户手工写入的笔记。它需要能够接收来自不同场景的知识材料，并把它们转换成后续可以理解、切分、追溯和学习的统一形态。

进入系统后，原始材料会被转换成两种形态：

```text
HTML
  用于预览和人工查看；

Markdown
  用于后续知识构建。
```

可以概括为：

```text
外部材料
  -> 转换为 HTML 和 Markdown
  -> 保留来源和上下文
  -> 进入知识单元构建
  -> 成为长期记忆的候选材料
```

## 1. 为什么需要外部知识输入

知识大脑的长期记忆来自外部世界。

这些外部知识最初并不一定是规整文档。

它们可能来自：

```text
PDF；
Word；
网页；
搜索引擎结果；
博客；
论文；
代码仓库；
IM 聊天记录；
会议纪要；
邮件；
用户笔记；
Agent 执行过程；
人工整理材料。
```

如果每种来源都直接进入后续知识处理，系统会很难统一理解它们。

因此，外部知识输入的第一职责，是把不同来源的材料转换成统一、可预览、可阅读、可追溯的 HTML 和 Markdown 内容。

## 2. 为什么同时生成 HTML 和 Markdown

HTML 和 Markdown 承担不同职责。

HTML 用于预览。

它尽量接近用户查看材料时的阅读形态，方便人工检查、回看和确认上下文。

Markdown 用于知识构建。

它是知识大脑的输入中间形态。

原因是：

```text
它能保留标题、段落、列表、引用、代码块和表格等结构；
它比纯文本更能表达作者原有组织；
它比复杂文档格式更容易被后续处理；
它便于人工阅读和修正；
它可以作为知识单元切分的结构基础。
```

知识单元不是直接从任意原始格式中生成，而是从规范化后的 Markdown 中生成。

这样可以让后续流程更稳定。

## 3. 外部知识输入要保留什么

外部知识输入不只是做格式转换。

它还需要保留材料的来源上下文。

例如：

```text
这段内容来自哪里；
是谁产生的；
什么时候产生的；
它属于什么场景；
它是否来自搜索结果；
它是否是聊天记录中的一部分；
它是否经过人工整理；
它和原始位置之间如何对应。
```

这些信息不会直接变成知识结论，但会影响后续判断。

一个来自个人长期笔记的判断，和一个来自搜索结果片段的判断，可信方式不同。

一段聊天记录中的观点，和一篇正式文档中的原则，使用方式也不同。

## 4. 不同来源如何进入系统

### 文档类材料

文档类材料通常已经有比较清晰的结构。

例如 PDF、Word、Markdown、网页文章、论文、博客。

系统需要尽量保留它们的标题层级、段落关系、列表、表格、引用和代码块。

这些结构会帮助后续知识单元判断边界。

### 搜索引擎材料

搜索引擎结果通常不是长期知识，而是为当前问题提供外部证据。

它们进入系统时，需要保留查询意图和来源页面。

搜索结果不应该直接被当成稳定长期记忆。

它们可以先作为证据材料参与临时认知模型。如果后续被反复验证有价值，再沉淀为长期记忆。

### 网页材料

网页材料可能包含正文、导航、广告、评论、脚注和更新时间。

系统应尽量保留与知识相关的正文和必要上下文，同时避免把无关页面噪音当成知识。

网页内容还需要特别关注时效性，因为网页可能更新、删除或过期。

### IM 聊天记录

聊天记录往往是碎片化、上下文依赖强、多人参与的材料。

它们进入系统时，需要保留对话上下文：

```text
谁说的；
什么时候说的；
前后讨论主题是什么；
哪些内容是问题；
哪些内容是回答；
哪些内容是决策；
哪些内容只是临时想法。
```

聊天记录不应该被简单当成正式知识。

它更适合作为经验、决策过程、问题线索或待整理材料。

### Agent 执行材料

Agent 在完成任务时也会产生知识材料。

例如：

```text
问题分析；
工具调用结果；
代码修改原因；
失败尝试；
最终解决路径；
用户反馈。
```

这些材料可以成为知识大脑学习的重要来源。

但它们同样需要经过整理，区分事实、推理、尝试和结论。

## 5. 外部知识输入与知识单元的关系

外部知识输入负责把外部材料变成预览 HTML 和规范化 Markdown。

知识单元从 Markdown 中形成。

两者的边界是：

```text
外部知识输入关注材料如何进入系统；
知识单元形成关注材料如何变成长期记忆的基础单位。
```

外部知识输入不急于判断最终知识结构。

它只需要让材料变得清晰、可读、可追溯，并保留足够上下文，让后续知识单元构建能够可靠发生。

## 6. 外部知识输入与临时认知模型的关系

并不是所有外部材料都会直接进入长期记忆。

有些外部材料只是为了回答当前问题而查找。

例如，搜索引擎结果、临时网页、工具返回内容，都可能先进入临时认知模型。

它们的作用是补充当前问题的证据，而不是立即成为稳定记忆。

如果这些材料在后续被确认重要、可靠、可复用，才适合进入长期记忆。

## 7. 总结

外部知识输入是知识大脑的外部感知入口。

它把不同来源、不同格式、不同可信程度的外部材料，统一转换为用于预览的 HTML 和用于知识构建的 Markdown。

一句话总结：

```text
外部知识输入不是简单导入文件，而是把外部世界转换成知识大脑可以继续理解和学习的材料。
```
