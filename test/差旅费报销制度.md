# 第一章 总则

第一条为适应公司经营发展需要，增强市场竞争力，有效降低成本，控制费用，

加强财务管理，提高公司员工办事效率，特制订本制度。

第二条差旅费指出差期间往返目的地城市所发生的交通费、住宿费、餐补费及其他费用，本制度适用于“万户软件”全体员工。

# 第二章 乘坐交通工具的要求

第三条交通工具的选择：

(一)火车：因公务出差乘坐火车的，夜间乘车在4小时以下，白天乘车时间在10小时以下，只能乘坐硬座席；夜间乘车在4小时以上，白天乘车在10小时以上的，可以乘坐卧铺车；（注：晚上十时起至次日凌晨六

时止为夜间乘车）。如因特殊原因需要另行处理的，必须事先向部门经理或上级领导申请，报分管副总经理批准，未经批准一律不予报销。乘坐高铁、动车时，仅可乘坐二等座。

(二)长途汽车：员工出差时如需乘坐长途汽车，需选择各种公共长途客车，

包括大巴、中巴、小巴和普通客车。不得以出租车作为长途交通工具。

(三)轮船：因公务出差乘坐轮船的，一般只能乘坐四等舱，超过船舱等级的必须事先向部门经理或上级领导申请，报分管副总经理批准，未经批准一律不予报销。

(四)因公用车：因特殊原因，需使用公司车辆出差的，须经过分管副总经理同意并按流程审批，且只报销油费与过路过桥费，不得另行报销出差地的市内交通费。为员工安全考虑，不得使用私家车作为出差交通工具。

(五)飞机：在以下两类情况下，可在易快报中发起“出差乘坐飞机申请流程”，流程批准后可以乘坐飞机：第一，紧急情况急需赶赴客户现场处理事务，在流程中选择“紧急情况”并说明理由，流程审批完成后，财务据此予以报销费用；第二，机票因季节性折扣导致飞机票总费用（含飞机票经济舱费用+保险+往返机场交通费）低于高铁二等票价格的，或者低于上述交通工具标准报价的（含无直达火车而选择转车的），需要差旅费报销制度在流程中列明相关费用，财务人员本着就低的原则处理报销费用，机票报销以行程单作为报销凭证。

第四条费用报销：

报销实际发生费用。如由客户承担或其他未实际发生的情况不予报销。

# 第三章出差期间发生的住宿费、交通费与伙食补贴

第五条住宿费

住宿费用采用限额实报实销制，即根据目的地城市划分标准在限额内实报实销。

(一)住宿酒店限连锁快捷酒店或城市快捷酒店，报销需提供正规合法的增值税专用发票，如遇特殊情况，可提前联系财务。

(二)公司在出差地有宿舍的，必须到宿舍入住，特殊情况下必须事先经分管副总经理批准，方可入住酒店，且报销时必须由分管副总经理陈述原由并签字方可报销。

(三)住宿限额标准按如下上限内实报实销，超标准住宿的，超出部分自行承

担：

| 分类   | A 类城市 | B 类城市 | C 类城市 | D 类城市 |
| ---- | ----- | ----- | ----- | ----- |
| 全体员工 | 350元  | 280元  | 220元  | 200元  |

(四)城市分类：

1.A 类城市：一线城市：北京、上海、广州、深圳；
2.B 类城市：新一线城市（15个），具体名单如下：成都、杭州、重庆、武汉、西安、天津、苏州、南京、郑州、长沙、东莞、沈阳、青岛、合肥、佛山；

3.C 类城市：二线城市（30个），具体名单如下：宁波、昆明、福州、无锡、厦门、济南、大连、哈尔滨、温州、石家庄、泉州、南宁、长春、南昌、贵阳、金华、常州、惠州、嘉兴、南通、徐州、太原、珠海、中山、保定、兰州、台州、绍兴、烟台、廊坊；

4.D 类城市：除 A、B、C 类城市以外的其他所有城市。
差旅费报销制度(五)同部门同行同性人员必须选择标准间住宿，按照两两优先入住原则，不得各自单独开设额外房间，否则取消所有补贴，并视情节轻重给予罚款。

(六)实际住宿费用超标不补，节约可以奖励：单人间奖励节约部分的50%，

标准间节约部分平均分配。

第六条交通费用

(一)出差地交通费用采用实报实销制度，且提交的票据必须是正规合法的交通票据。

(二)原则上出差期间往返均应乘坐公共交通，即公交车、地铁与机场大巴等。

(三)以下情况可以乘坐出租车或网约车：

1.自办公地出发，交通工具出发时间在9点之前，到达时间在21点以后的，可以报销车站往返的出租车或网约车的的士费用，其他时间原则上以公共交通为准。

2、异地需要乘坐出租车或网约车的，必须持机打发票或电子发票（定额发票无效），且发票所示时间必须和出差时间相吻合，打车事由在合理的范畴内。报销时，持出租车机打发票必须在背面注明来往起始点、到达地单位全称及需要打车的事由，非机打或电子发票、发票时间与出差时间不吻合、未注明打车事由的票据均不予报销。各级主管需要核实异地打车起始点发生费用的合理性，对于虚列费用的事项一经查实，除追回报销人费用以外，对直接主管与中心负责人按照虚报费用的两到五倍予以罚款并通报。

第七条伙食补贴

(一)伙食补贴按照出差起始时间、采用日期算头不算尾的方式计算补贴天数，

对实际出差时间在24小时以上，超过8小时的加记1天。

(二)补贴标准

| -------- | ---------- | ---- | --- | ------------------------------------------------------------------- |

差旅费报销制度吉林、辽宁、云南、青海、宁夏、甘肃、内蒙。（上述偏远省份就地招聘的人员，省内出差按照原标准不上浮）第八条费用报销及补贴标准确定原则(一)住宿费用按照标准限额内实报实销，超标准住宿的，超出部分个人承担。

(二)住宿费用报销必须凭酒店开具的增值税专用发票方可办理报销手续，因此入住时必须先落实该酒店的出具票据情况。

(三)投亲靠友办理住宿的，按照出差地住宿标准的40%予以补贴。其他情况无法提供发票的不得报销住宿费用。

(四)多人同时入住一人付款取得发票的，其他人员不需要提供住宿票据，财务人员核实确认后直接按单据发放餐补。

(五)餐补费用的发放天数按照财政部门所规定的“算头不算尾”的原则计算

具体天数，对于实际出差时间在一天（24小时）以上、对其超过8小时以外时间按加记1天发放补贴。

(六)出差期间的交通费用采用实报实销制度，提交的票据必须是正规合法的交通票据。出租车票必须是机打发票，发票所记载的打车时间必须和出差时间相吻合，且必须在背面注明来往起始点与打车事由。

(七)如因外地客户或者伙伴提供服务出差，公司如与客户有协议，由客户承担住宿的情况，只发放伙食补贴与实报实销交通费用。

(八)公司外派学习、培训、开会的，住宿及伙食均在培训费及会务费中的，

不再发放任何形式的补贴；如只包含住宿的，按照会议所在地城市发放伙食补贴。

(九)当天往返超过12小时的按40元/次予以伙食补贴；当天往返不足12小时的按30元/次予以伙食补贴。

# 第四章 报销要求

第九条出差人员以从简开支的原则预计出差费用，通过易快报中“借款及款项支付流程”，向公司申请借款。出差过程中需保管好可报销费用相关的纸质或电子发票，出差回单位后，应及时在易快报中发起相应的报销流程：项目核算差旅费报销或非项目核算差旅费报销，如果前期有借款未还，在报销时需要在流程中选择“核销借差旅费报销制度款”，报销金额不足冲抵借款的，需在财务审核后，按照借款余额向公司归还借款，报销金额大于借款部分，费用审核后，财务按差额支付给员工。

第十条乘坐任何交通工具报销时，都须持有相关的凭证、单据，否则不予报销。

第十一条特殊情况下，超过标准、超过报销期限的差旅费用必须经过公司总经理同意并在书面单据签字后，财务方可办理支付手续，无总经理签字的单据不得提前支付。

# 第五章 说明

第十二条出差期间因公发生的招待、礼品等费用等不属于差旅费范围，员工必须提前报知部门经理或上级领导、经分管副总经理批准后方可办理，并另行报销。

第十三条有条件乘坐卧铺车而未乘坐的，可以按同种票面金额出差的60%给予奖励，计算公式为：（同程卧铺价—同程座铺价）*60%。

第十四条出差期间如公事处理完毕后无公共交通工具（火车、长途汽车）可供选择时，若住宿费用＞包车费用，员工视实际情况可包车出行，须事先向分管副总经理申请，未经批准的一律不予报销。
